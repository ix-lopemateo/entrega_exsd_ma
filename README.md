# entrega_exsd_ma
Este repositorio contiene los materiales y resultados del Trabajo de Explotación Semántica de Datos (ESD), cuyo objetivo es la construcción y explotación de un grafo de conocimiento RDF que represente las relaciones entre microbiota intestinal, metabolitos microbianos, procesos epigenéticos y fenotipos de adiposidad temprana en el eje madre–infante.

El trabajo se basa en información procedente de la literatura biomédica y, en particular, en el estudio prospectivo de la cohorte española NEMO, y emplea tecnologías de la Web Semántica para integrar y consultar conocimiento heterogéneo de forma estructurada e interoperable.
